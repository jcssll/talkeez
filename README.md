# Talkeez AAC — Free & Accessible Communication Board  
Talkeez AAC is a free, open‑source communication board designed to support children, caregivers, educators, and families who need simple, reliable tools for everyday communication.

This project powers **Talkeez.org/aac**, a lightweight AAC board built using **HTML, CSS, and JavaScript**. It is intentionally simple, fast, and accessible so anyone can use it without cost, login, or installation.

---

## 🌟 Mission

Talkeez AAC exists to make communication tools accessible to everyone — especially families and communities who struggle with structure, routine, or access to specialized resources.

We believe:

- Every child deserves a voice  
- Every caregiver deserves support  
- Every educator deserves tools that work  
- Essential communication should never be behind a paywall  

As we continue researching and listening to our communities, we will release **additional free items** that help support communication, structure, and daily routines.

---

## 🧩 Features

### ✔ Free AAC Board  
A simple, tap‑based communication board with essential words and phrases.

### ✔ Mobile‑Friendly  
Works on phones, tablets, laptops, and assistive devices.

### ✔ No Login Required  
Instant access — no account, no subscription, no tracking.

### ✔ Offline‑Friendly  
Because it’s built with HTML, CSS, and JS, it can run even without internet once loaded.

### ✔ Community‑Driven  
We add new free items based on real needs from families, teachers, and homeschool communities.
